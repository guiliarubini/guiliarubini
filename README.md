# Giulia Rubini - Personal Website

> Personal portfolio website for Guilia Rubini (Ruby) - Fashion Designer 

### Technology Stack

- React, TypeScript, Tailwind CSS, Vite, React Icons + Font Awesome + ESLint

## Deployment View

### 7.1 Build Process

Deployment is configured by [deploy.yml](.github/workflows/deploy.yml) file via `npm`. 

### 7.2 Makefile Commands
To build and run locally for development
```makefile
make              # build and run
```

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation & Running

```bash
git clone https://github.com/BondaiKa/ruby_website.git && \
cd ruby_website && \
make
```
